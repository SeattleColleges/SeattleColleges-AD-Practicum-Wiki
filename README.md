## Projects Overview

- <span style="color:#1E90FF;"><strong>NSC Events</strong></span>  
A platform for North Seattle College students to discover, register, and track campus events. The web and mobile apps, developed with the NSC Application Development team, consolidate event information, notifications, and participation tracking to help planners organize successful events.

- <span style="color:#FF69B4;"><strong>Belindas Closet</strong></span>  
  A not-for-profit platform providing North Seattle College students with professional clothing for interviews, employment, and graduation. Web and Android apps, backed by a shared backend, help staff manage inventory and expand services across the Seattle Colleges system.

- <span style="color:#32CD32;"><strong>Pelletier Construction Group</strong></span>  
  Pelletier Construction Group specializes in home remodeling and construction projects in Seattle, offering innovative design and building solutions. Expertise ranges from open concept kitchens to large-scale renovations and new constructions.

- <span style="color:#FF8C00;"><strong>GeekGirlCon</strong></span>  
  Mobile application for GeekGirlCon attendees to access event information easily. This is a private repository; access can be requested via email or Slack. Active repos include GeekGirlCon Mobile and GeekGirlCon Backend.

- <span style="color:#8A2BE2;"><strong>Licton Springs Review</strong></span>  
  A website and mobile app showcasing visual and literary art from NSC students, staff, faculty, and alumni. Completely redesigned from WordPress, the platform offers an immersive experience with seamless web and mobile access.

- <span style="color:#00CED1;"><strong>AD Program Website</strong></span>  
  The central hub for showcasing current and completed student projects in the NSC Application Development program. Highlights student contributors, supports mentorship, and demonstrates real-world tech solutions for potential clients.

- <span style="color:#DC143C;"><strong>Shift App (Expo)</strong></span>  
  A mobile-first application helping work-study students manage their shifts. Features include picking up, dropping, and trading shifts, user authentication, and real-time updates. Focused on building a solid MVP with Expo and Supabase for backend services.
